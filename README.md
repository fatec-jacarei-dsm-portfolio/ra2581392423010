### Ⅰ. Sobre Mim ✦
Estudante de **Desenvolvimento de Software Multiplataforma**, atualmente no **4º semestre**. Com um sólido background em logística, trago uma visão analítica e organizada para o desenvolvimento de soluções tecnológicas complexas. Atuei como **Product Owner**, gerenciando sprints e alinhando stakeholders em projetos reais.

Sou apaixonado por interfaces elegantes, arquitetura limpa e experiências que encantam, atuando tanto na stack **MERN** quanto no desenvolvimento mobile com **React Native**.
